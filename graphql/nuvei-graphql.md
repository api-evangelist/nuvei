# Nuvei GraphQL API

Nuvei is a global payment technology company. The API covers payment processing, payment methods, refunds, payouts, subscriptions, fraud management, 3DS authentication, and transaction reporting across multiple currencies and regions.

**Endpoint:** No native GraphQL endpoint — conceptual schema derived from Nuvei REST API
**Documentation:** https://docs.nuvei.com

- Reference: https://docs.nuvei.com/documentation/integration/server-to-server/server-to-server-introduction/
- Schema: https://github.com/api-evangelist/nuvei
